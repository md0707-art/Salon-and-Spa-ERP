# Salonix ERP System

## Overview

Salonix is a comprehensive Salon & Spa ERP (Enterprise Resource Planning) system developed using Laravel, PHP, MySQL, Blade, Tailwind CSS, and JavaScript.

The system is designed to streamline salon and spa operations by integrating customer relationship management, appointment scheduling, inventory management, point of sale (POS), reporting, and business analytics into a single platform.

---

## Features

### Customer Relationship Management (CRM)

* Customer registration and profile management
* Customer history tracking
* Customer engagement management

### Appointment Management

* Appointment booking
* Schedule management
* Service assignment
* Conflict detection

### Point of Sale (POS)

* Billing and invoicing
* Transaction management
* Payment tracking

### Inventory Management

* Product inventory tracking
* Purchase management
* Stock monitoring
* Inventory alerts

### Reporting & Analytics

* Sales reports
* Service reports
* Inventory reports
* Business performance analytics

### User Management

* Role-based access control
* Staff management
* Authentication and authorization

## Installation

```bash
git clone https://github.com/md0707-art/salonix.git
cd salonix

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```
